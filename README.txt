MANUAL DO SISTEMA HOST - DESIGN INSPIRADO NA GDOOR
==================================================

Este manual foi redesenhado seguindo o layout e estilo do site da Gdoor (https://gdoor.com.br),
mantendo a identidade da Alphatech Sistemas e todas as funcionalidades originais.

CARACTERÍSTICAS DO NOVO DESIGN:
===============================

1. HEADER MODERNO
   - Navegação responsiva com menu hamburger para mobile
   - Logo da Alphatech Sistemas
   - Hero section com call-to-action
   - Gradiente azul inspirado na Gdoor

2. SEÇÕES BEM DEFINIDAS
   - Cards com ícones FontAwesome
   - Seções alternadas (branco/cinza claro)
   - Títulos e subtítulos bem estruturados
   - Grid responsivo

3. CARDS INTERATIVOS
   - Hover effects com elevação
   - Ícones coloridos com gradiente
   - Animações de entrada
   - Modal com informações detalhadas

4. FOOTER COMPLETO
   - Informações de contato da Alphatech
   - Links para redes sociais
   - Horário de atendimento
   - Design profissional

5. RESPONSIVIDADE
   - Menu mobile com hamburger
   - Grid adaptativo
   - Textos e imagens responsivos
   - Touch-friendly

FUNCIONALIDADES:
================

- Navegação suave entre seções
- Modal com informações detalhadas de cada funcionalidade
- Menu mobile funcional
- Animações e transições suaves
- Links diretos para WhatsApp
- Design moderno e profissional

ARQUIVOS INCLUÍDOS:
===================

- Manual_Completo_Sistema_Host.html (HTML principal)
- manual_host_style.css (Estilos CSS)
- manual_host_script.js (JavaScript)
- images/ (Pasta com imagens)
- README.txt (Este arquivo)

COMO USAR:
==========

1. Abra o arquivo Manual_Completo_Sistema_Host.html no navegador
2. Navegue pelas seções usando o menu superior
3. Clique nos cards para ver informações detalhadas
4. Use o menu mobile em dispositivos menores
5. Entre em contato via WhatsApp clicando no link

TECNOLOGIAS UTILIZADAS:
=======================

- HTML5 semântico
- CSS3 com Flexbox e Grid
- JavaScript ES6+
- FontAwesome para ícones
- Google Fonts (Inter)
- Design responsivo

adaptando elementos como:
- Paleta de cores azul
- Layout de cards
- Hero section
- Footer estruturado
- Tipografia moderna
- Gradientes e sombras
